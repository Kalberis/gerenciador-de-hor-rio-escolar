# Gerenciador de Horário Escolar

Este projeto é um sistema web desenvolvido com Django para gerenciar horários escolares, incluindo cadastro de turmas, professores, disciplinas e horários.

## Como executar

1. Ative o ambiente virtual:
   ```
   venv\Scripts\activate
   ```
2. Instale as dependências:
   ```
   pip install -r requirements.txt
   ```
3. Execute as migrações:
   ```
   python manage.py migrate
   ```
4. Inicie o servidor:
   ```
   python manage.py runserver
   ```

## Funcionalidades
- Cadastro de turmas
- Cadastro de professores
- Cadastro de disciplinas
- Cadastro e visualização de horários

## Estrutura
- Projeto Django: `horario_escolar`
- App principal: `core`
